## Why VaniMMo?
- Vani is like a shortened word for vanilla
- There are many features
- Quick, easy
## What will it support?
- Clan: parties, create a clan's own area
- Boss: powerful creatures
- Stats: statistics table
- Pet: pet
- Skill: skills
- Item: Customize item attributes
